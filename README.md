# 📖 About

**SIGMA POS Mobile** is a modern Android Point of Sale application developed to simplify sales transaction management.

The application enables cashiers and business owners to manage products, perform transactions, monitor sales, and view reports directly from an Android device.

# 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/USERNAME/sigma-pos-mobile.git
```

Open using Android Studio.

Sync Gradle.

Run the application.

# ☁️ Supabase Configuration

Create a `Constants.java` file.

```java
public class Constants {

    public static final String SUPABASE_URL =
            "YOUR_SUPABASE_URL";

    public static final String SUPABASE_KEY =
            "YOUR_SUPABASE_ANON_KEY";

}
```
